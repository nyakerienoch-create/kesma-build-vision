# Kesma Contractor's and Suppliers Website Development Plan

Create a professional, modern, and high-end construction website for Kesma Contractor's and Suppliers.

## 1. Project Setup
- Update `index.html` with relevant SEO meta tags.
- Define theme colors (Primary: Dark Slate/Navy, Secondary: Professional Gold/Safety Orange accent, White/Light Gray background).
- Setup Lucide icons and Framer Motion.

## 2. Page Sections
- **Navbar**: Sticky navigation with scroll-to-section.
- **Hero Section**: High-impact background image, headline "Building Excellence, Delivering Trust", and a CTA button "Request a Quote".
- **About Section**: Professional bio of Kesma Contractor's and Suppliers as a general contractor.
- **Services (Menu Highlights)**: Grid of key services (General Contracting, Residential Building, Commercial Construction, Interior Renovation).
- **Testimonials**: Customer reviews showcasing reliability and quality.
- **Gallery**: Visual showcase of past projects (3-column grid).
- **Contact Section**: Phone (0722 838336), Location (Moi Ave, Bungoma), and contact form.
- **Footer**: Company links and social icons.
- **WhatsApp Button**: Floating button for instant booking.

## 3. Technical Requirements
- Use React 19 and Vite.
- Responsive design for mobile, tablet, and desktop.
- Framer Motion for scroll-triggered animations.
- Lucide-React for professional iconography.
- Shadcn UI for form components.
- Meta description for SEO.

## 4. Design Guidelines
- Modern typography (Inter or Montserrat).
- 8pt grid system.
- Professional imagery.
- Clear call-to-actions.
